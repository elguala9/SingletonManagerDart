# singleton_manager_test

Comprehensive test suite and utilities for the `singleton_manager` package.

## Purpose

This package contains:
- Unit tests for singleton manager core functionality
- Functional tests for features and use cases
- Integration tests for end-to-end scenarios
- Test fixtures and helpers

## Usage

This package is **not published on pub.dev**. It's used internally within the monorepo for testing.

### Running Tests

```bash
# From monorepo root
melos run test

# Or in this package
dart test
```
